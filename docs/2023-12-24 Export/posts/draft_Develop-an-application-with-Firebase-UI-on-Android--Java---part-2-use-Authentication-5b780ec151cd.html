Develop an application with Firebase UI on Android (Java): part 2 use Authentication

This is part 2 of an article series about developing an app using Firebase UI on Android (Java). We are going to authenticate an user by his Email address and password or his Google account. Firebase/UI allows for a lot more authentication provider but we are concentrating on these two (see the Firebase Auth docs for more information).

Thanks to Firebase UI Auth we do not need to hassle about layouts for the authentication part, we just need to start the process and Firebase UI handles all for us. In the end our user is signed in the authentication database (don’t mix this with “real” databases we are talking in the next articles). If the user has not signed in before the user is asked to provide a user name beneath his Email and password.
